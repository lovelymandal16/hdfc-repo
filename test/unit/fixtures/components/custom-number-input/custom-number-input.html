Examples of Custom Number Input Formats

Currency (INR)

Format: ₹#,##0.00

Example: ₹1,234.56

Description: Indian Rupee with thousands separator and 2 decimal places

Currency (USD)

Format: ¤#,##0.00

Example: $1,234.56

Description: US Dollar with thousands separator and 2 decimal places

Plain Currency

Format: ¤####0.00

Example: $1234.56

Description: Currency without thousands separator

Percentage

Format: #,###,##0%

Example: 12,345%

Description: Percentage with thousands separator

3 Decimal Places

Format: #,###,##0.000

Example: 1,234.567

Description: Number with thousands separator and 3 decimal places
